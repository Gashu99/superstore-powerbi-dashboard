# 📊 Superstore Sales Dashboard — Power BI

🇫🇷 Tableau de bord interactif sur Power BI, basé sur le dataset Sample Superstore.

🇬🇧 Interactive Power BI dashboard based on the Sample Superstore dataset.



## 🇫🇷 Pages

- **Vue Globale** — CA, Profit, Marge, Commandes avec comparaisons vs N-1, évolution mensuelle et répartition par catégorie
- **Analyse Clients** — segmentation, CA moyen par client, top clients
- **Analyse Produits** — rentabilité, marge par sous-catégorie, produits à marge négative

## 🇬🇧 Pages

- **Global Overview** — Revenue, Profit, Margin, Orders with year-over-year comparisons, monthly trend and category breakdown
- **Customer Analysis** — segmentation, average revenue per customer, top customers
- **Product Analysis** — profitability, margin by sub-category, products with negative margin

---

## Outils / Tools

Power BI · DAX · Power Query

## Exemple de mesure DAX / DAX measure example


Croissance % =
DIVIDE([CA Total] - [CA Mois Precedent], [CA Mois Precedent])
